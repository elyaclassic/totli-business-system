# Server boshqaruv — hammasi bitta faylda

**Ochish:** Ikki marta bosing — **`SERVER_BOSHQARUV_OCHISH.bat`** (oyna barqaror ochiladi).  
Yoki to‘g‘ridan-to‘g‘ri **`server_boshqaruv.bat`** ni ishga tushiring.

Barcha ishlar uchun **bitta asosiy fayl**: **`server_boshqaruv.bat`**. Ochilsa menyu chiqadi:

1. **Holatni tekshirish** — server ishlayaptimi yoki yo‘qmi (port 8080 band/bo‘sh).
2. **To‘xtatish** — serverni to‘xtatish (port 8080 ni band qilgan protsess o‘chiriladi).
3. **Qayta ishga tushirish** — avval to‘xtatadi, keyin orqa fonda `start_server_fon.bat` orqali qayta ishga tushiradi.
4. **Chiqish** — dasturni yopish.

---

## Qisqacha

| Nima qilmoqchisiz     | server_boshqaruv.bat ichida |
|-----------------------|-----------------------------|
| Holatni tekshirish    | 1 ni tanlang                |
| To‘xtatish            | 2 ni tanlang                |
| Qayta ishga tushirish | 3 ni tanlang                |

Port 8080 ni boshqa ishlatsangiz — `server_boshqaruv.bat` ni Notepad da ochib yuqoridagi `set PORT=8080` qatorini o‘zgartiring.
